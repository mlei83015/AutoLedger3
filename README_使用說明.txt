AutoLedger V19

本版新增：
1. 新增第三種桌面小工具：圖片＋載具＋記帳小工具。
2. 圖片小工具可在設定中選擇自訂圖片，並調整圖片區域大小。
3. 圖片小工具保留載具條碼、餘額、今日花費、支出與收入按鈕；移除「點我輸入金額」那一格。
4. 三條線側邊選單新增「欠款紀錄」，可記錄誰欠你錢，也可以扣還款，例如欠 1000、還 100 後剩 900。
5. 設定頁底部版本顯示 AutoLedger V19。

安裝後建議：
1. 到設定 → 圖片小工具設定，先選擇小工具圖片與圖片區域大小。
2. 到桌面新增「圖片＋載具＋記帳」小工具。
3. 點三條線 → 欠款紀錄，新增你要追蹤的欠款。
