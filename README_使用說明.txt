AutoLedger 自動記帳 V24

本版重點：
1. 圖片桌面小工具鎖定圖片區塊，避免小工具被拉高時圖片被拉長或壓扁。
2. 三條線選單新增「匯率換算」。
3. 匯率換算支援常用幣種：TWD、USD、JPY、KRW、CNY、MYR、GBP、EUR、HKD。
4. 上下兩格都能輸入金額，會自動換算另一格。
5. App 會每天嘗試更新一次匯率；若無網路或更新失敗，會先使用內建參考匯率。
6. V23 的 LINE Pay 點數折抵偵測、跨來源防重複、自動記帳、桌面小工具、備份與 CSV 都保留。

注意：
- 匯率換算只是參考值，實際刷卡金額、海外手續費仍以銀行入帳為準。
- 圖片小工具如果已經在桌面上被拉得很高，建議移除後重新新增一次 V24 小工具，版面會比較穩。
