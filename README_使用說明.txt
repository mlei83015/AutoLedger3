AutoLedger V16：分類標題、圖標選擇與圓角主題優化版

本版重點：
1. 自動通知記帳只保留金額、收支方向與原始通知，不強制用 LINE 錢包或發票載具當標題。
2. 首頁最近紀錄改成以「分類」當主要標題；分類改成茶飲，列表就會顯示茶飲。
3. 修改紀錄頁新增圖標選擇，可選茶飲、餐飲、交通、禮物、遊戲、醫療、學習等常用圖標。
4. 圖標會跟著分類顯示，也可在修改紀錄時手動更換。
5. 彈窗、按鈕、輸入框與卡片加大圓角，減少方方正正的感覺。
6. 介面顏色會依深色 / 淺色模式調整。
7. 防重複記帳、跨來源偵測、CSV 匯出、備份、還原、清除資料都保留。

安裝後建議：
1. 到設定確認通知讀取來源與防重複功能是否開啟。
2. 到設定看通知讀取用途說明，再開啟通知讀取權限。
3. 測試 LINE Pay、載具、Google 錢包與銀行通知是否會正確記帳。
