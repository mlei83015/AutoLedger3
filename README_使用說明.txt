AutoLedger 自動記帳 Version 3

這版是依照新版 UI 設計做的 Android 自用版。

主要功能：
1. 首頁：剩餘餘額、本月圓形圖、今天總花費、最近紀錄。
2. 手動新增：支出 / 收入、金額、分類、店家/來源、備註/用在哪裡、常用項目、最近使用。
3. 設定：LINE Pay、載具發票、銀行通知、簡訊通知、其他 App 可各自開關。
4. 修正：不再讀取「自動記帳」自己跳出的通知，避免重複記帳。
5. 防重複記帳：同一來源、同金額、短時間內的通知會自動合併。
6. 特色：智慧分類、昨日花費摘要、預算提醒、Google 同步備份預留入口。

安裝後要先做：
1. 打開 App。
2. 按「開啟通知讀取權限」。
3. 在系統設定裡允許「自動記帳通知讀取」。
4. 若 Android 顯示「受限制設定」，請長按 App 圖示 > 應用程式資訊 > 右上角三點 > 允許受限制設定。

更新方式：
把這份 ZIP 上傳 GitHub，使用 Codespaces 解壓縮後 push，Actions 會產生 AutoLedger-debug-apk。
