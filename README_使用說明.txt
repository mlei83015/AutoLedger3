AutoLedger V15：智慧分類留白與修改介面優化版

本版新增與修正：
1. 自動通知記帳不再自動填入分類與來源，分類、來源/店家會先保持空白，方便你之後自己點進去修改。
2. LINE Pay、載具、Google 錢包、銀行通知的防重複邏輯仍會讀原始通知內容，因此分類留白不會影響同筆偵測。
3. 修改紀錄彈窗重新設計，改成深色圓角卡片，文字更清楚、輸入框更好看。
4. 點分類欄位時會出現小型快速分類列，例如茶飲、飲料、午餐、晚餐、交通、超商、訂閱；收入則有薪水、零用錢、打工、退款等。
5. 手動新增頁的分類欄位也支援小型快速分類列。
6. 設定頁最下方顯示目前版本：AutoLedger V15。
7. 介面卡片、按鈕與輸入框圓角再加大，減少方方角角的感覺。

更新方式：把本 ZIP 解壓縮到 GitHub 專案根目錄，提交後等待 Actions 產生 APK。
