AutoLedger V29

本版修改：
1. 修正「圖片＋載具＋記帳」桌面小工具：
   - 新增自適應版型（一般 / Tall / Compact），依桌面小工具高度切換。
   - 開機後會自動重新更新三種小工具，降低系統更新或重開機後圖片變形的情況。
   - 圖片區塊重新放大，讓顯示更穩定、不容易變成過扁長條。
2. 本月預算圓形圖改成更平面的圓弧風格，保留圓角收尾。
3. 首頁本月財務狀況補上第三條資訊：「剩餘預算」。
4. 側邊欄刪除「發票記帳 HOT」與「共享帳本」。
5. 新增「固定收支」功能：
   - 可新增 / 編輯 / 刪除固定收支項目。
   - 可單筆新增，也可一次套用全部固定收支。
6. 新增「財務模擬」功能：
   - 可輸入本月預算、已花費、額外收入、預計再花。
   - 即時顯示模擬後剩餘金額與使用率。
7. 原本匯率換算、欠款紀錄、記帳小工具、分類管理、功能設定等功能保留。
