AutoLedger 自動記帳 V11 修正版

本版新增與修正：
1. 使用者提供的淺藍色圖示已套用為 App 桌面圖示。
2. 桌面小工具重新美化，背景、按鈕與間距更清爽。
3. 載具＋記帳小工具只顯示一個條碼，不再重複顯示載具號碼。
4. 點桌面小工具上的載具條碼可複製手機條碼號碼。
5. 鈴鐺通知設定彈窗文字對比提高，深色模式下也能清楚閱讀。
6. 設定載具條碼彈窗 UI 調整，輸入框尺寸與對齊修正。

注意：Android 桌面小工具不能直接放真正可輸入文字的輸入框，所以快速記帳仍使用「點我輸入金額」開啟快速新增小視窗。


修正版：修正鈴鐺通知時間格式驗證造成 GitHub Actions 編譯失敗的問題。


AutoLedger V12 穩定版更新：
1. 支出金額全面改為紅色，收入維持綠色，餘額/結餘更容易辨識。
2. 最近記錄重新整理：標題、副標題、金額、時間固定對齊，長文字自動省略。
3. 發票載具、LINE Pay、Google 錢包、銀行刷卡通知顯示更乾淨，不再把整段通知塞滿列表。
4. 加強同日同金額跨來源防重複：LINE Pay / 載具 / Google 錢包 / 銀行通知同金額時，優先判定同一筆。
5. 首頁新增本月記錄筆數：顯示自動記帳與手動記帳數量。
6. 小工具支出按鈕改紅色、收入按鈕改綠色，視覺規則和 App 一致。

V13 更新：
- 桌面小工具快速新增彈窗改成更小、更圓角的快速輸入卡片。
- 快速新增只保留金額與分類兩欄，移除來源欄位，操作更快。
- 快速常用項目改成更小的固定 3 欄網格，不會因為換行或不同字數而歪掉。
- 快速常用項目不再帶入固定金額，金額一律由使用者自行輸入。
