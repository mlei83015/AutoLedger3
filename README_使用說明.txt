AutoLedger V23 自動記帳

這是 Android 自用測試版：
- 通知自動記帳：LINE Pay、載具發票、Google 錢包、銀行通知、簡訊通知。
- 手動新增收入 / 支出。
- 桌面小工具：簡易記帳、載具＋記帳、圖片＋載具＋記帳。
- CSV 匯出、JSON 備份 / 還原、清除資料、錯誤回報與自動除錯紀錄。

V23 新增：
1. LINE Pay 點數 / 優惠折抵偵測。
   - 如果通知內有「原價 / 折抵 / 實付」，App 會用實付金額記帳。
   - 例如原價 $30、點數折抵 $5、實付 $25，首頁與預算會扣 $25。

2. LINE Pay 原價通知與銀行 / Google 錢包實扣通知交叉比對。
   - 如果 LINE Pay 先跳 $30，但銀行或 Google 錢包跳 $25，App 會判斷可能是同一筆。
   - 會自動合併成一筆，保留原價 $30、折抵 $5、實付 $25。
   - 可在「紀錄詳情」看到原價、折抵與實際金額。

3. 防重複仍保留。
   - LINE Pay、載具、Google 錢包、銀行通知同一筆消費，不會重複記兩次。

使用方式：
1. 安裝 APK。
2. 到 Android 設定開啟「通知讀取權限」。
3. 回 App 設定通知偵測來源。
4. 如果自動判斷不準，點紀錄可查看，長按或點修改可更正金額、分類、來源與備註。

注意：
- App 只能讀取通知文字，不能直接登入 LINE Pay、銀行或載具帳號。
- 如果通知沒有顯示實付金額，也沒有銀行 / Google 錢包通知可比對，App 只能先用通知上的金額記帳，之後可手動修改。
- 目前資料預設存在手機本機。
