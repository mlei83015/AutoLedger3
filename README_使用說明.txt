AutoLedger V27 使用說明

這版新增：
1. 點首頁上方「剩餘餘額」可以設定目前實際還剩多少錢。
2. 本月預算與目前餘額會連動：如果本月預算 $10,000，但現在只剩 $5,000，圓形圖會自動換算為已花費 $5,000、剩餘 $5,000。
3. 「設定本月可花預算」仍在設定頁的本月預算區，預算本身不會因設定目前餘額而被改掉。
4. 快速常用項目、線上匯率、桌面小工具、防重複、CSV、備份、還原都保留。

安裝方式：
1. 把專案上傳到 GitHub Codespaces。
2. Terminal 執行：gradle :app:assembleDebug
3. APK 會在 app/build/outputs/apk/debug/app-debug.apk

注意：
匯率來源為 open.er-api.com。實際刷卡或換匯仍以銀行、信用卡公司或交易平台入帳匯率為準。
