AutoLedger 自動記帳 V25

本版重點：
1. 匯率換算頁改成「台幣 1 元可換多少外幣」的顯示方式，九宮格更清楚。
2. 幣種選擇視窗改成可滑動清單，常用幣種放最上面，並新增更多常見國家幣別。
3. 選取幣種時顏色改得更明顯，避免深色模式下看不清楚。
4. 圖片桌面小工具鎖定圖片比例，避免小工具被拉高時圖片被拉長或壓扁。
5. 上下兩格都能輸入金額，會自動換算另一格。
6. App 會每天嘗試更新一次匯率；若無網路或更新失敗，會先使用內建參考匯率。
7. V23/V24 的 LINE Pay 點數折抵偵測、跨來源防重複、自動記帳、桌面小工具、備份與 CSV 都保留。

注意：
- 匯率換算只是參考值，實際刷卡金額、海外手續費仍以銀行入帳為準。
- 圖片小工具如果已經在桌面上被拉得很高，建議移除後重新新增一次小工具，版面會比較穩。
