AutoLedger V17

本版重點：
1. 補齊設定頁與管理頁尚未更新的彈窗，包含收入分類管理、支出分類管理、常用項目管理、還原資料、錯誤回報、預算設定、刪除確認、清除資料等，都改成更圓弧的卡片式視窗。
2. 彈窗內的輸入框、按鈕、內容區塊一併改圓角，整體 UI 風格更一致。
3. 保留 V16 的分類標題顯示、圖標選擇、深淺色模式與防重複記帳功能。
4. 設定頁底部版本文字簡化成 AutoLedger V17。

安裝後建議：
1. 到設定確認通知讀取來源與防重複功能是否開啟。
2. 到設定查看通知讀取用途說明，再開啟通知讀取權限。
3. 測試 LINE Pay、載具、Google 錢包與銀行通知是否會正確記帳。
