AutoLedger 自動記帳 Version 5

這版是針對實際試用後的問題修正：

V5 重要更新：
1. 首頁最上方「剩餘餘額」改成全部餘額：本月預算 + 全部收入 - 全部支出。
   所以手動新增收入後，上方餘額會增加。
2. 圓形圖改成只看「本月可花預算」與「本月支出」。
   收入不會讓圓形圖預算變大，只會加到上方全部餘額。
3. 深色模式下，圓形圖中間百分比文字改成亮色，不會看不到。
4. 快速常用項目與最近使用改成固定網格排列，不會因為文字行數或項目數量歪掉。
5. 自動除錯 / 防重複記帳加強：LINE Pay 先記、載具發票較晚進來，如果金額相同且屬於付款/發票來源，會自動判斷同一筆，不再重複新增。
6. 保留排除自己通知：自動記帳 App 自己發出的通知不會被再記一次。

主要功能：
- 自動讀取 LINE Pay、載具發票、銀行通知、簡訊通知、其他付款 App 通知。
- 手動新增支出 / 收入，可輸入金額、分類、店家/來源、備註。
- 會記住最近使用與常用項目，方便下一次快速點選。
- 可設定本月可花預算。
- 可切換深色模式與圖表配色。
- 有昨日花費摘要、預算提醒、智慧分類、Google 同步備份預留入口。

安裝後要先做：
1. 打開 App。
2. 按「開啟通知讀取權限」。
3. 在系統設定裡允許「自動記帳通知讀取」。
4. 若 Android 顯示「受限制設定」，請長按 App 圖示 > 應用程式資訊 > 右上角三點 > 允許受限制設定。

GitHub Actions 打包：
把這份 ZIP 上傳 GitHub，使用 Codespaces 解壓縮後 push，Actions 會產生 AutoLedger-debug-apk。
