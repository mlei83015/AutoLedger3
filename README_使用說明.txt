AutoLedger V22 自動記帳

這是 Android 自用測試版：
- 通知自動記帳：LINE Pay、載具發票、Google 錢包、銀行通知、簡訊通知。
- 手動新增收入 / 支出。
- 新增收入時可勾選「加回本月可用預算」，適合中獎、獎金、額外收入。
- 本月預算會每個月重新計算；額外加回只記在當月。
- 桌面小工具：簡易記帳、載具記帳、圖片＋載具＋記帳。
- 圖片小工具支援從手機相簿選圖，並裁切固定 4:3 顯示區塊。
- 小工具圖片點一下可回到 App 修改圖片。
- 欠款紀錄：可記錄誰欠你錢，也能扣還款。
- 匯出 CSV、備份 JSON、還原資料、清除資料。

使用方式：
1. 安裝 APK。
2. 開啟通知讀取權限。
3. 到設定開啟想偵測的通知來源。
4. 如需桌面小工具，長按桌面加入 AutoLedger 小工具。

注意：
資料預設儲存在手機本機，不會自動上傳。


V22：圖片小工具照片圓角貼齊卡片，載具條碼只顯示條碼，號碼另外顯示；手動新增頁縮小、備註改彈窗、快速常用項目改成橫向滑動。
