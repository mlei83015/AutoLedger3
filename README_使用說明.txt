AutoLedger 自動記帳 V10

新增重點：
1. 新增兩種 Android 桌面小工具：
   - 簡易記帳小工具：顯示全部餘額、今日花費、快速支出/收入。
   - 載具＋記帳小工具：上方顯示載具條碼，下方顯示餘額、今日花費、快速支出/收入。
2. 設定頁新增「設定載具條碼」。輸入自己的手機條碼號碼後，桌面小工具會自動產生條碼。
3. 小工具新增「點我輸入金額」入口，會開啟快速新增小視窗。
4. App 圖示改成淺藍色、簡潔記帳風。
5. 保留 V9：餘額、預算圖、財務報表、側邊選單、鈴鐺通知設定、防重複記帳、自動排除自己通知等功能。

注意：
Android 桌面小工具本身不能直接放真正的打字輸入框，因此 V10 採用「點小工具輸入欄 → 開快速新增小視窗 → 儲存同步回 App」的方式。
