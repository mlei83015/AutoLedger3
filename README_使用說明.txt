AutoLedger V30

本版主要修改：
1. 桌面「載具 + 自動記帳」小工具改成固定版型，避免重開機或重新加入後切換版型造成圖片看起來變形。
2. 小工具圖片區域改成自動撐滿剩餘空間，底下收入 / 支出 / 載具區塊鎖在下方，減少底部空白。
3. 圖片輸出改成穩定 4:3 高解析裁切，使用 centerCrop，不拉伸原圖。
4. 圖標選擇新增更多常用項目。
5. 圖標選擇加入「＋新增圖標」，可以自己輸入 emoji 與名稱。
6. 設定頁底部顯示 AutoLedger V30。

安裝方式：
1. 上傳 ZIP 到 Codespaces。
2. 依照 AutoLedger_V30_Codespaces_指令.txt 貼上指令。
3. GitHub Actions 成功後，到 Actions 的 artifact 下載 APK。
